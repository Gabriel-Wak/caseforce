# 🎁 Case Force

**Case Force** é um jogo web inspirado em sistemas de abertura de caixas, inventário de skins, XP, batalhas e contratos.  
O projeto foi criado com foco em uma experiência visual moderna, organizada e divertida, sem uso de dinheiro real.

> ⚠️ Este projeto é apenas para fins de estudo, portfólio e entretenimento.  
> Não envolve apostas, depósitos reais ou qualquer tipo de dinheiro de verdade.

---

## 🚀 Sobre o Projeto

O **Case Force** é uma plataforma onde os jogadores podem abrir caixas gratuitas, ganhar XP, evoluir de nível, desbloquear novas caixas, participar de batalhas e montar um inventário com itens virtuais.

A ideia principal é simular a experiência de um jogo de abertura de caixas, mas de forma segura, gratuita e totalmente voltada para aprendizado e demonstração de habilidades em desenvolvimento web.

---

## ✨ Funcionalidades

- 🎁 Sistema de abertura de caixas
- ⭐ Sistema de XP e níveis
- 🔓 Caixas desbloqueáveis por nível
- 🧰 Inventário de skins virtuais
- 🔄 Sistema de contratos
- ⚔️ Sistema de batalhas de caixas
- 👤 Sistema de login para jogadores reais
- 🏆 Ranking dos jogadores com inventário mais valioso
- 🔎 Filtros de caixas por preço, raridade e tipo
- 📊 Área de vitórias recentes
- 🎨 Interface inspirada em sites modernos de games
